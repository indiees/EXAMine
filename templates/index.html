Dev note - there's a few front end frameworks already enabled here, we have a set of icons we can access from fontawesome,
and a bunch of styles already defined in styles.css, which you might want to look through to see if anything's helpful.
as for the structure of the frontend, the app is launched through main.py, which just loads up all the other pages defined in tools/pages.py, which is a definition of just each endpoint by itself
any helper methods for performing general logic or aws integration are in tools/helpers.py, and any decorator style functions are in tools/decorators.py
all pages are loaded within templates/index.html as a flask template that fill the rest of the container below this message.

The rest of the code in this project at the time of writing is just templating leftover from my A2 so feel free to change things
Also make sure to setup your keys from the IAM account I generated for you using "aws configure" if you haven't already or else everything will be broke
